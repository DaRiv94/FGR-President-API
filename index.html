Welcome to FGR President API!

FGR President API is an example api that uses Microsoft Azure Table Storage

Key Features

  • Uses Microsoft Azure Table Storage
  • Covers Create Read Update Destory (CRUD) operations

Technologies

  • Node.js - As javaScript runtime
  • Express.js - As web framework on Node.js to help with running the server
  • Azure Table Storage - For storing data in the cloud
  • JavaScript - As programming language used thoughout
  • Jest - For testing
  • Git - For source control
  • GitHub - For external repository
  • Heroku - For cloud hosting
